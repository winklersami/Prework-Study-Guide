HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata is the information that helps the browser render correctly.
  • HTML stands for Hypertext Markup Language.
  • HTML is the foundation for any webpage that contains the structural formatting.
  • HTML is made up of different tags enclosed within brackets that tell the computer how to format the page.
  • Finally, HTML is made up of several elements that delineate the type of content within that webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheet, and controls the style of the webpage.
  • There are 3 types of CSS: Inline, Internal, and External.
  • Inline CSS is embedded into the HTML code that already exists while Internal CSS Code formats sections within the HTML code.
  • External CSS code is made in a separate document that is then linked in the head of an HTML file.

Git

  • git status: checks what branch we are currently in.
  • git checkout -b 'branch name': Creates a new branch with the corresponding 'branch name' and switches to it.
  • git pull origin main: pulls up the most up to date version of the main branch.
  • git checkout main: switches to the main branch.
  • git add -A: commits all files in the working branch.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Javascript helps add functionality to a webpage.
  • Strings are a specific type of variable that lists words. They are contained inside quotation marks.
  • A variable can contain strings, numbers, boolean (true/false), and many other types of data!
  • An array is a type of variable that contains multiple different data sets.
